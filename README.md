# Asteroids - Boot.dev

This is a Python-based implementation of the classic arcade game **Asteroids**. It was developed as part of the [Boot.dev](https://www.boot.dev) curriculum to master object-oriented programming, game loops, and 2D graphics.

## Project Overview

The goal of this project is to build a fully functional game using the **Pygame** library. This project covers:

- Implementing a game loop (Process input -> Update state -> Render).
- Using Vector math for movement and physics.
- Object-Oriented Programming (OOP) for game entities like Players, Asteroids, and Bullets.
- Collision detection and game state management.

## Tech Stack

- **Language:** Python 3.13
- **Library:** Pygame
- **Package Manager:** [uv](https://github.com/astral-sh/uv)

## How to Run

1. Ensure you have `uv` installed.
2. Clone the repository and navigate to the project folder.
3. Install dependencies and run the game:
   ```bash
   uv sync
   uv run main.py
   ```
